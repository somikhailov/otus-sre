Создание диаграммы с помощью Diagrams

В данном разделе описан процесс создания диаграммы для проекта c помощью python скрипта. Больше о проекте MKdocs можно узнать - https://www.mkdocs.org/

Установка

pip install diagrams

Использование

переходим в каталог diagrams и запускаем скрипт

cd diagrams/
python3 diagram.py
